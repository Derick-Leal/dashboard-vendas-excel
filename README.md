# 📊 Dashboard de Vendas para E-commerce em Excel

Projeto desenvolvido em Microsoft Excel para gerenciamento e análise de vendas de um e-commerce. A planilha centraliza informações de produtos, vendedores e vendas, permitindo o acompanhamento de indicadores por meio de dashboards interativos.

## 📋 Funcionalidades

- Cadastro de produtos
- Cadastro de vendedores
- Registro de vendas
- Histórico de vendas
- Consultas de informações
- Dashboards gerenciais
- Tabelas dinâmicas para análise dos dados

## 📁 Estrutura da Planilha

| Aba | Descrição |
|------|-----------|
| **Menu** | Página inicial para navegação pela planilha. |
| **Produtos** | Cadastro dos produtos comercializados. |
| **Vendedores** | Cadastro dos vendedores. |
| **Vendas** | Registro das vendas realizadas. |
| **Vendas Histórico** | Histórico consolidado das vendas. |
| **Consultas** | Área destinada à consulta de informações. |
| **Dashboard** | Indicadores e métricas atuais de vendas. |
| **Dashboard Histórico** | Evolução histórica dos indicadores. |
